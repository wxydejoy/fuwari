# 摄影作品目录

## 使用说明

1. 直接将您的摄影作品图片上传到这个目录下
2. 支持的图片格式：JPG、PNG、WebP 等常见格式
3. 图片会自动在摄影页面展示，不需要修改任何配置文件
4. 图片会根据实际比例自动排列，实现美观的瀑布流布局
5. 图片名称会自动转换为可读的标题（例如："my_photo.jpg" → "My Photo"）

## 最佳实践

- 建议使用高质量图片，但注意文件大小，避免影响页面加载速度
- 图片名称建议使用英文、数字、下划线或连字符，避免使用特殊字符
- 定期整理图片，删除不需要展示的图片

## 示例

直接将图片文件（如：sunset.jpg、mountain_view.png 等）拖放到此目录即可。
